# VulnForge

Autonomous AI agent for HackerOne bug bounty hunting.

## Project Structure

- `scanner/`: Core scanning engine (Python).
- `backend/`: API and database management (Node.js/TypeScript).
- `cli/`: Command-line interface for hunters (Node.js).
- `reports/`: Generated vulnerability reports.

## Getting Started

### Prerequisites

- Node.js (v18+)
- Python (v3.10+)
- Poetry (for Python dependency management)
